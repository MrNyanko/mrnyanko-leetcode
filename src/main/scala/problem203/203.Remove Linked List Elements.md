## Remove Linked List Elements

Remove all elements from a linked list of integers that have value val.

**Example:**
```
Given: 1 --> 2 --> 6 --> 3 --> 4 --> 5 --> 6, val = 6
Return: 1 --> 2 --> 3 --> 4 --> 5
```

---

## 删除链表中的元素

删除链表中等于给定值 val 的所有元素。

**示例：**
```
给定： 1 --> 2 --> 6 --> 3 --> 4 --> 5 --> 6, val = 6
返回： 1 --> 2 --> 3 --> 4 --> 5
```
